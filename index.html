Resumo - Atalhos para o VS Code

Atalhos gerais úteis

  • Alt + tab — Alterna entre aplicativos abertos
  • Ctrl + y — Refaz uma ação / o oposto do ctrl + z
  • Tecla Windows + ← ou → — Limita a exibição da página na tela a posicionando na direita ou esquerda, possibilitando a utilização de duas telas abertas ao mesmo tempo
  • Tecla Windows + ↑ — Desfaz a duplicação e maximiza a tela atual
  • Tecla Windows + ↓ — Minimiza a tela atual

Atalhos gerais de computação

  • Tecla home — Faz o cursor ir para o início da linha atual
  • Tecla end — Faz o cursor ir para o fim da linha atual
  • Shift + end — Seleciona todo o conteúdo que esteja entre o cursor e o final da linha atual
  • Shift + home — Seleciona todo o conteúdo que esteja entre o cursor e o início da linha atual
  • Ctrl + ← ou → — Move o cursor para a palavra seguinte ou para a palavra anterior
  • Ctrl + shift + ← ou → — Seleciona a palavra seguinte ou a palavra anterior, podendo ser usado para selecionar múltiplas palavras
  • Ctrl + shift + ↑ ou ↓ — Seleciona todo o bloco de linhas na direção desejada, podendo ser usado para selecionar múltiplas linhas ao mesmo tempo
  • Ctrl + backspace — Exclui a próxima palavra à esquerda
  • Ctrl + del — Exclui a próxima palavra à direita
  • Ctrl + h — Exibe a caixa de diálogo Substituir para localizar e substituir palavras, partes de textos ou textos completos
  • Ctrl + f — Busca por uma palavra-chave/texto dentro do arquivo atual que estiver aberto

Atalhos para o VS Code

  • Ctrl + ] — Divide a tela do VS Code
  • Ctrl + shift + p — Abre um campo para pesquisa de atalhos dentro do VS Code
  • Ctrl + o — Abre janela de diálogo para que seja selecionado um arquivo a ser aberto
  • Ctrl + shift + p — Abre a aba configurações do VS Code
  • Ctrl + n — Abre uma nova aba de arquivos
  • Ctrl + shift + n — Abre uma nova janela do VS Code
  • Ctrl + shift + w — Fecha a janela atual do VS Code
  • Ctrl + b — Mostra ou esconde a tela lateral “explorador-workspace”
  • Ctrl + shift + enter — Insere uma linha acima
  • Ctrl + shift + k — Exclui a linha atual em que está o cursor
  • Shift + alt + ↑ ou ↓ — Duplica uma linha de código inteira
  • Ctrl + g — Redireciona a tela para uma linha de código especificada
  • Tag + * + número de tags a serem criadas — Cria o número escolhido de uma mesma tag sem a necessidade de criar uma por uma
  • Alt + ↑ ou ↓ — Move a linha para baixo ou para cima
    Dica: Para esse atalho o VS Code considera cada elemento (tag e seu conteúdo) como sendo uma única linha
  • Ctrl + L — Seleciona toda a linha em que o cursor estiver posicionado
  • Ctrl + p — Busca um arquivo pelo nome e o abre
  • Ctrl + f2 — Seleciona todas as correspondências da palavra atual e atribui a elas múltiplo cursor para que possam ser editadas ao mesmo tempo
  • Ctrl + shift + L — Faz a mesma ação do atalho ctrl + f2
  • Ctrl + shift + f — Abre a guia de pesquisa para pesquisar palavras- chaves dentro de todo o projeto
  • Ctrl + shift + e — Abre a guia explorador-workspace
  • Ctrl + ; — Comenta e “descomenta” todo o código que estiver selecionado
    Dica: Para comentar apenas um único elemento (tag e seu conteúdo) não precisa fazer a seleção, basta pressionar o atalho com o cursor estando em meio ao elemento
  • Shift + alt + f — Auto formata todo o arquivo de código
